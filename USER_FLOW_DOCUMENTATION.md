# OfferIQ User Flow Documentation

# AI-First Flow: Intelligence Report → Automated System

# Date: April 10, 2026

# Version: 2.0 - Streamlined

## EXECUTIVE SUMMARY

This document outlines the streamlined user journey where AI analysis creates the foundation for everything.
The intelligence report drives sales copy, funnel structure, page generation, and email sequences.

## USER PERSONA

Sarah Chen - Small business owner selling a $497 online course on "Social Media Mastery"

- Tech-savvy but not a developer
- Wants automated lead generation and sales
- Budget: $500/month for ads
- Goal: 50 sales/month

---

## PHASE 1: OFFER ANALYSIS (FOUNDATION)

### Screen 1: Offer Input & AI Analysis

**URL**: `/analyze`
**Purpose**: User inputs offer, AI generates comprehensive intelligence report

**UI Elements**:

- Header: "Tell us about your offer"
- Offer Details Form:
  - Product/Service name
  - Price point
  - Target audience
  - Key benefits (3-5 points)
  - Unique selling proposition
- "Analyze My Offer" button

**User Action**: Sarah enters her course details and clicks analyze

### Screen 2: Intelligence Report Generation

**URL**: `/report/generating`
**Purpose**: AI analyzes offer through structured phases

**UI Elements**:

- Multi-phase progress:
  - Phase 1: "Structural Analysis (scoring, frameworks, matrices)..."
  - Phase 2: "Strategic Analysis (psychology, positioning, narratives)..."
- Live insights appearing:
  - "Calculated offer score: 72/100"
  - "Identified 5 pain points"
  - "Recommended lead magnet funnel"
- Report preview snippets

**Background Processing**:

- **Call 1 (Sonnet)**: Structural intelligence
  - Offer scoring and matrices
  - Funnel blueprint and pricing strategy
  - Platform priority analysis
- **Call 2 (Opus)**: Strategic intelligence
  - Persona psychology and positioning
  - Conversion hooks and messaging angles
  - Monetization strategy narrative

### Screen 3: Intelligence Report Dashboard

**URL**: `/report/social-media-mastery`
**Purpose**: Review AI-generated insights that drive everything else

**UI Elements**:

- **Structural Intelligence Tab**:
  - Offer score breakdown (market viability, audience clarity, etc.)
  - Funnel structure blueprint
  - Revenue model architecture
  - Platform priority matrix

- **Strategic Intelligence Tab**:
  - Persona psychological profile
  - Positioning analysis and repositioning strategy
  - Conversion hook library
  - Messaging angle matrix

- **Monetization Strategy Tab**:
  - Product core value perception
  - Real-world use case scenarios
  - Master monetization narrative

- **Action Buttons**:
  - "Generate Funnel" (uses structural blueprint)
  - "Create Sales Copy" (uses strategic hooks/angles)
  - "Build Email Sequence" (uses persona psychology)
  - "Generate Traffic Intelligence" (uses platform priorities)

**User Action**: Sarah reviews the comprehensive report, impressed by the depth, clicks "Generate Complete System"

**Constraint**: Cannot proceed without accepting the intelligence report

---

## PHASE 2: AI-GENERATED SYSTEM (AUTOMATED CREATION)

### Screen 4: System Generation Progress

**URL**: `/generate/system`
**Purpose**: AI creates complete marketing system based on intelligence report

**UI Elements**:

- Multi-step progress:
  1. "Creating optimized funnel structure..."
  2. "Writing conversion-focused sales copy..."
  3. "Designing high-converting landing pages..."
  4. "Building automated email sequences..."
  5. "Setting up traffic intelligence..."

- Live preview panels showing each component being created
- "Pause & Edit" option for each step

**User Action**: Sarah watches the system build itself, occasionally pausing to tweak headlines

### Screen 5: System Review & Customization

**URL**: `/system/review`
**Purpose**: Review and make final adjustments to the AI-generated system

**UI Elements**:

- **Funnel Overview**: Visual flow with conversion estimates
- **Page Gallery**: Thumbnail previews of all landing pages
- **Email Timeline**: Sequence overview with subject lines
- **Traffic Recommendations**: AI-suggested budget allocation

- **Quick Edit Panel**:
  - Change headlines/colors
  - Adjust pricing
  - Modify email send times
  - Update traffic budget

**User Action**: Sarah makes minor tweaks to branding colors and email timing

---

## PHASE 3: LAUNCH & OPTIMIZATION

### Screen 6: Pre-Launch Checklist

**URL**: `/launch/checklist`
**Purpose**: Ensure everything is ready for launch

**UI Elements**:

- Checklist items:
  - [ ] Payment processor connected
  - [ ] Email service linked
  - [ ] Domain connected
  - [ ] Traffic budget allocated
- "Launch System" button (disabled until all checked)

**User Action**: Sarah connects Stripe and Mailchimp, clicks launch

### Screen 7: Live Dashboard

**URL**: `/dashboard/live`
**Purpose**: Monitor the running system

**UI Elements**:

- Real-time metrics:
  - Leads captured today
  - Sales this week
  - Email open rates
  - Traffic sources performance

- System Health:
  - Funnel conversion rates
  - Email sequence performance
  - Traffic campaign ROI

- Quick Actions:
  - "Pause Traffic" / "Resume"
  - "Edit Email Sequence"
  - "Optimize Funnel"

**User Action**: Sarah monitors initial results, sees first sales coming in

## STREAMLINED FLOW COMPARISON

### Removed Steps:

- ❌ Manual funnel template selection
- ❌ Empty canvas page building
- ❌ Separate email sequence creation
- ❌ Complex traffic setup forms

### Enhanced Steps:

- ✅ AI-driven intelligence report as foundation
- ✅ One-click system generation
- ✅ Integrated review and quick edits
- ✅ Automated launch preparation

### Key Product Logic Integration:

1. **Intelligence Report First**: Everything stems from deep offer analysis
2. **AI Generation**: No manual creation - AI builds complete system
3. **Report-Driven Copy**: Sales copy uses psychological insights from report
4. **Automated Sequencing**: Email timing based on customer psychology data
5. **Smart Traffic**: Budget allocation uses market intelligence

---

## CONSTRAINTS & BUSINESS RULES

### Hard Constraints:

1. Must complete offer analysis before any generation
2. Intelligence report must be reviewed before system creation
3. Cannot launch without payment/email integration
4. System generation cannot be skipped

### AI-Driven Automation:

- Funnel structure based on offer type and market data
- Sales copy uses identified pain points and benefits
- Email sequences incorporate psychological triggers
- Traffic recommendations based on budget and market analysis

### User Control Points:

- Can pause/edit during generation
- Can modify headlines and key copy
- Can adjust email timing and traffic budget
- Can pause/resume live campaigns
